# Cloudinary Image Uploader

## Project Overview
This project is a **Node.js backend service** for uploading images directly to Cloudinary.  
It allows you to upload images via **API endpoints** without storing them locally. The uploaded images are saved in a Cloudinary folder, and the API returns the **secure URL** and **public ID** for each image.

---

## Features
- Upload images to Cloudinary directly from API requests  
- No local storage required  
- Returns image URL and public ID for easy access  
- Simple Node.js + Express setup  

---

## Tech Stack
- Node.js  
- Express.js  
- Multer (for handling multipart/form-data)  
- Cloudinary
